Solicitud de crédito de consumo Centaurus

Los créditos rotativos generalmente permiten realizar compras o tranferencias electrónicas y telefónicas. Algunos bancos también permiten hacer uso de estos con una chequera o una tarjeta débito.

Beneficios:
- Se puede utilizar como y cuando se necesite.
- Permite decidir qué cantidad del cupo se utiliza (total o parcial).
- Los pagos realizados por el cliente liberan el crédito nuevamente.
